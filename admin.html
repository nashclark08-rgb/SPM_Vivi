📺 Online Teacher Status

Teachers in online meetings can scan a QR code, enter their name, and press a button — their name then appears live on the main display screen and the parent QR screen, alerting parents not to interrupt. Requires a free Firebase Realtime Database.

Teachers scan this QR code on PTI night to set their online meeting status.

Setup steps:
1. Create Firebase project & Realtime Database
2. Set rules: ".read":true, ".write":true
3. Paste database URL above & Save
4. Teachers scan this QR on PTI night
5. Names appear live on the display screen
